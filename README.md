# lojinha
Este projeto consiste no desenvolvimento de um sistema web para controle de estoque de um pequeno comércio (“Lojinhainha”), criado como um trabalho escolar. A aplicação foi desenvolvida utilizando Python com o framework Flask e banco de dados MySQL. 

O sistema permite o gerenciamento completo de informações essenciais, incluindo cadastro e listagem de clientes, fornecedores, vendedores, produtos em estoque e registro de vendas. Além disso, possui funcionalidades de exclusão de registros e organização dos dados por meio de relacionamentos entre tabelas no banco de dados.

A interface foi construída com HTML, CSS e Bootstrap, buscando oferecer uma navegação simples e intuitiva, com menu lateral e páginas organizadas para cada funcionalidade. O principal objetivo do projeto é facilitar o controle e a organização de um estoque de forma prática, centralizando informações importantes e auxiliando na gestão do negócio.
